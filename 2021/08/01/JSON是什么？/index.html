JSON是什么？

维基百科对JSON的概括是这样的
JSON（JavaScript Object Notation，JavaScript对象表示法，读作/ˈdʒeɪsən/）是一种由道格拉斯·克罗克福特构想和设计、轻量级的资料交换语言，该语言以易于让人阅读的文字为基础，用来传输由属性值或者序列性的值组成的数据对象。尽管JSON是JavaScript的一个子集，但JSON是独立于语言的文本格式，并且采用了类似于C语言家族的一些习惯。

资料交换语言，这是JSON的核心概念，但仅仅这样还不足以理解JSON，所以可以举个例子

假设现在有一个服务器和客户端，它们之间需要进行数据的收发，但是在数据收发时使用的函数recv()和send()，它们并不知道它们发/收的数据是什么，它们只是将一段线性内存中的数据做拷贝并且通过TCP/IP发送到另一端，那么我们如何才能知道我们发/收的数据的含义呢？
说白了，就是在服务端和客户端之间需要有一个约定好了的一种规定，我们可以根据这种规定从收到的数据中得到我们想要的东西，最简单地说，就像我们要发数据，先约定好双方都用同一种结构体，一端将一个结构体发到另一端，而另一端因为知道发来的这一段数据是双方约定好的结构体，所以可以用结构体来解析它，最终得到自己想要的数据，JSON就是这么一种东西。

在我看来，JSON和结构体可以说很像，它是结构体的一种上层抽象，让数据交换的格式脱离编程语言的限制，成为了一种范式
